# Word Grouping App

Hosted @ https://oriontl.github.io/word-grouping-app/

## Local Development

Clone the repo

`git clone git@github.com:oriontl/word-grouping-app.git`

Install dependencies

`npm i`

Spin up the local server @ [http://localhost:3000](http://localhost:3000)

`npm start`

Run tests

`npm test`
